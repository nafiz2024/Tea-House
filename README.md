# Tea House Website

<img src="./Tea House.png">

## Overview

**Tea House** is a modern and responsive website showcasing a tea shop and its products. The website features banners, featured products, client testimonials, news sections, and a stylish footer with subscription options. Built using **HTML, Tailwind CSS, and Font Awesome**, it provides a clean, professional, and user-friendly interface.

## Features

- Fully responsive design with mobile and desktop support
- Banner section with high-impact title, description, and call-to-action button
- Featured Products section with product cards
- Great Tea section highlighting premium quality and unique taste
- Client Testimonials section with layered card layout
- News section showcasing updates and promotions
- Footer with quick links, services, help section, newsletter subscription, and social media icons
- Tailwind CSS and Font Awesome integration for modern styling and icons

## Technologies Used

- HTML5
- Tailwind CSS
- Google Fonts (Manrope)
- Font Awesome Icons
- Images and graphics for banners, products, and clients
